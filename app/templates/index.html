NeuralMusic is a web application that generates MIDI piano melodies.
It is powered by a neural network, trained for tens of hours on music famous human artists.

Be aware of the following limitations:
· The application only support monophonic MIDI generation.
· Every timestep contains a note or a chord. There are no breaks. Hence, it sounds a bit odd.
· The generated sequences are not visualized in the browser, but they can be played directly on the webpage.
· There is no cherrypicking here, so...
... Some generated sequences are “more interesting” than others.
